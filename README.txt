# Max Blocker

Утилита для блокировки "MAX" на Windows.

## Что блокирует:
- ❌ Процессы `max.exe`, `max-service.exe`
- ❌ Домены MAX в сети (max.com, api.max.com, update.max.com)
- ❌ Установщики MSI, связанные с MAX
- ❌ Повторную установку через автозагрузку

## Как работает:
- Завершает процессы MAX при запуске
- Добавляет домены в файл hosts (127.0.0.1)
- Маскируется под системные процессы
- Работает из автозагрузки

## Установка:
1. просто запустите от администратора либо он сам

## Важно:
- Требуются права Администратора
- Работает на Windows (мы не знаем на каких лучше думаем 10 11
- антивирус может ругаться если что!
